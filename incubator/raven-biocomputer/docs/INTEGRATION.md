# Ecosystem integration

- **Raven AI**: consumes evidence-linked run receipts.
- **JSpace Chain**: supplies policy-gate and structured-reflection envelopes.
- **Home for AI**: displays local run state, artifacts, replay status, and audit records.
- **Hermes Edge**: routes deterministic tools locally before larger models.
- **OpenClinical AI**: accepts only reviewed translational outputs; clinical decisions remain human-controlled.

The standalone repository also contains a stable MCP v1 adapter, FastAPI surface, Hugging Face Gradio app, JSON schema, signed-artifact roadmap, and pluggable future isolation backends.
