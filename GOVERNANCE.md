# Governance

This project is currently maintainer-led.

## Maintainer responsibilities

- Keep the public roadmap clear.
- Review security-sensitive changes carefully.
- Require tests or rationale for infrastructure changes.
- Preserve the open-source, reproducible, evidence-linked direction of the Raven ecosystem.

## Decision principles

1. Scientific and clinical claims need evidence boundaries.
2. Local-first and affordable deployment matter.
3. Auditability beats hidden magic.
4. Security issues take priority over feature work.
5. The Raven ecosystem should stay coherent rather than splinter into overlapping products.
