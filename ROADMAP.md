# Roadmap

## North star

Build affordable, inspectable, open-source AI infrastructure for agentic biology, healthcare deployment, and local orchestration.

## Near term

- Stabilize CI across Python, TypeScript, Rust, and documentation.
- Replace demo-only flows with explicit development gates.
- Add architecture diagrams and API reference snapshots.
- Expand reproducible examples and test fixtures.
- Improve issue triage and contribution workflow.
- Ship Raven Evidence Graph as the shared claim, source, confidence, risk, and trace contract across Raven apps.
- Add privacy-aware provider profiles for DeepSeek DSpark, local fallback, and future model adapters.

## Mid term

- Harden runtime authentication and tenant isolation.
- Publish versioned agent/workflow packs.
- Add model cards and provenance reporting.
- Add deployment recipes for local, cloud, and edge environments.
- Attach Evidence Graph traces to clinical, research, home-orchestration, and edge benchmark workflows.
- Wire provider profiles into live adapters with policy checks, cost telemetry, and Evidence Graph provider tags.

## Long term

- Make Raven AI a coherent open-source research platform for labs, researchers, classrooms, startups, and clinical teams.
- Make evidence provenance portable across local notebooks, hosted demos, clinical workflow substrates, and edge devices without locking users into one cloud provider.
- Support cheap-first remote inference and local-first edge inference as first-class deployment choices.
